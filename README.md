# TabKeeper - 智能标签页管理器

一个功能强大的 Chrome 浏览器扩展，用于管理和组织您的标签页。基于 OneTab 的理念，但添加了更多实用功能。

## ✨ 核心功能

### 🎯 基础功能（与 OneTab 相同）

- **一键保存** - 点击扩展图标，保存当前窗口所有标签页并关闭
- **批次管理** - 标签页按保存时间分组，每组可独立管理
- **快速恢复** - 点击恢复单个标签页或整个批次
- **编辑标题** - 为每个批次添加自定义标题
- **显示时间** - 每个批次显示创建时间

### 🚀 改进功能

#### 1. 性能优化

- ✅ **分页加载** - 每页仅显示 10 个批次，避免大量数据导致卡顿
- ✅ **懒加载** - 标签页列表支持滚动加载
- ✅ **实时统计** - 显示总标签页数量

#### 2. 分类管理 📁

- ✅ **自定义分类** - 创建分类如"软件项目"、"知乎文章"、"学习资料"等
- ✅ **批次分类** - 为每个标签页批次指定分类
- ✅ **分类筛选** - 按分类快速过滤显示
- ✅ **默认分类** - 新保存的批次默认为"未分类"

#### 3. 备份功能 💾

- ✅ **本地导出** - 将所有数据导出为 JSON 文件
- ✅ **本地导入** - 从备份文件恢复数据
- ✅ **数据格式** - 包含版本信息、导出日期、标签页组、分类等

#### 4. 云同步 ☁️

- ✅ **Chrome Sync** - 使用 Chrome Storage Sync API
- ✅ **多设备同步** - 登录同一 Google 账号的设备自动同步
- ✅ **同步限制** - 最多同步最近 50 个批次（云存储限制）
- ✅ **手动同步** - 点击"云同步"按钮立即同步

### 🔍 其他功能

- **搜索功能** - 搜索标签页标题或 URL
- **删除管理** - 删除单个标签页或整个批次
- **响应式设计** - 适配不同屏幕尺寸

## 📦 安装方法

### 开发者模式安装

1. **下载项目**

   ```bash
   git clone <repository-url>
   cd demo-project-onetab
   ```

2. **准备图标**（必需）
   - 在 `icons` 文件夹中放置以下图标：
     - `icon16.png` (16x16)
     - `icon48.png` (48x48)
     - `icon128.png` (128x128)
   - 或使用在线工具生成图标

3. **加载扩展**
   - 打开 Chrome 浏览器
   - 访问 `chrome://extensions/`
   - 启用右上角的"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择项目文件夹 `demo-project-onetab`

4. **开始使用**
   - 点击工具栏中的 TabKeeper 图标保存标签页
   - 打开新标签页即可看到 TabKeeper 界面

## 🎮 使用指南

### 保存标签页

1. 打开多个要保存的标签页
2. 点击浏览器工具栏中的 TabKeeper 图标
3. 所有标签页自动保存并关闭（保留 TabKeeper 页面）

### 管理批次

- **编辑标题** - 点击批次标题编辑
- **设置分类** - 点击分类标签更改分类
- **恢复全部** - 点击"恢复全部"按钮在新窗口打开所有标签页
- **删除批次** - 点击"删除"按钮删除整个批次

### 管理单个标签页

- **打开标签页** - 点击标签页标题
- **删除标签页** - 悬停时点击"删除"按钮

### 分类管理

1. 点击"📁 分类筛选"显示分类面板
2. 点击"+ 新建分类"创建自定义分类
3. 点击分类按钮筛选显示
4. 点击批次的分类标签更改分类

### 备份与恢复

- **导出备份** - 点击"💾 导出备份"下载 JSON 文件
- **导入备份** - 点击"📂 导入备份"选择备份文件

### 云同步

- **自动同步** - 保存标签页时自动同步到云端
- **手动同步** - 点击"☁️ 云同步"按钮
- **多设备** - 在其他设备登录相同 Google 账号即可同步

## 🔧 技术实现

- **Manifest V3** - 使用最新的扩展 API
- **Chrome Storage API** - 本地存储和云同步
- **Chrome Tabs API** - 标签页管理
- **Chrome Windows API** - 窗口管理
- **原生 JavaScript** - 无依赖，轻量高效

## ⚙️ 配置选项

当前版本包含以下默认配置：

- 恢复标签页后自动从列表删除
- 新窗口恢复（除非 TabKeeper 是唯一标签页）
- 每页显示 10 个批次
- 云同步最多 50 个批次

## 📝 注意事项

1. **图标准备** - 必须准备图标文件才能正常加载扩展
2. **云同步限制** - Chrome Sync 有存储大小限制（约 100KB）
3. **数据安全** - 定期导出备份防止数据丢失
4. **浏览器兼容** - 仅支持基于 Chromium 的浏览器（Chrome、Edge 等）

## 🐛 问题反馈

如遇到问题，请检查：

1. 是否启用了开发者模式
2. 图标文件是否存在
3. Chrome 版本是否支持 Manifest V3
